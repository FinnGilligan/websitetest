Computer Science Seminar Projects

Mars Rover Photo Search

Developed an iOS app using Swift and NASA’s public API to allow users to search and view photos taken by Mars rovers. Focused on UI design, API integration, and efficient handling of remote image data.

WeDialed – Productivity App

Created a productivity app in Swift where users could add friends and share progress. Designed and implemented the entire data structure and backend integration with Firebase, including authentication, user data storage, and a dynamic friends system.
